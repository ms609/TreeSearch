# Tree topology for matrix simulation

The tree topology used to generate the matrices in
[`congreveLamsdellMatrices`](https://ms609.github.io/TreeSearch/reference/congreveLamsdellMatrices.md)

## Usage

``` r
referenceTree
```

## Format

A single phylogenetic tree saved as an object of class `phylo`

## Source

Congreve & Lamsdell (2016);
[doi:10.1111/pala.12236](https://doi.org/10.1111/pala.12236)
